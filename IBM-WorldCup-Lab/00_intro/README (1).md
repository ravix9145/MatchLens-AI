# Build an AI-Powered World Cup Predictor with IBM Bob

## Overview

Welcome to the **AI in Sports: World Cup Predictions Learning Lab**.

In this lab, you will build a **machine learning application that predicts soccer match outcomes** using historical data — with the help of **IBM Bob**, your AI-powered coding assistant.

This is a **guided learning lab**, designed to help you:

* Learn by doing
* Build a real AI-driven application
* Explore how AI can support software development

You do not need to be an expert in Python or machine learning — **IBM Bob will assist you throughout the process**. 

***

## What you will learn

By completing this lab, you will learn how to:

* Use IBM Bob to generate and understand Python code
* Build a machine learning workflow using Jupyter notebooks
* Analyze and prepare real-world sports data
* Train and evaluate a prediction model
* Work with development tools such as Git repositories and notebooks

You will also gain experience using an **AI coding assistant to accelerate development**, a key skill in modern software engineering.

## How IBM Bob is used in this lab

IBM Bob acts as your **AI development partner**:

* Generates code based on your prompts
* Helps debug errors
* Explains concepts and workflows
* Assists with environment setup

Instead of writing code from scratch, you will:

1. Prompt Bob
2. Review the generated code
3. Run it inside the notebook
4. Observe and interpret the results

This approach lets you focus on **understanding the workflow and results**, rather than syntax and implementation details.
